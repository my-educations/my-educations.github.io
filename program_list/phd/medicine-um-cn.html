马来亚大学医学院-博士招生简章

2022年3月18日


1.学院简介


马来亚大学医学院自成立以来，已成功培训了6000名马来西亚医生和5000名医学专家，并照顾了许多生命。医学学院和马来亚大学医学中心保证继续其基金会的遗产，即由马来西亚人培养优秀的马来西亚医生。马来亚大学的历史始于1949年，当时爱德华七世国王医学院和莱佛士书院由马来亚卡尔·桑德斯大学教育委员会合并，以满足新加坡和马来亚联邦的需要。 1959年1月15日，马来亚大学改组成立了法院、中央议会、毕业生协会、总理、亲校长和副校长，并设有两个平等的自治区，一个在新加坡，另一个在吉隆坡。1962年1月，新加坡分校成为新加坡大学，并于1980年被称为新加坡国立大学或“国大”，而马来亚大学则留在吉隆坡，成为马来西亚第一大大学。 1960年5月，成立了一个研究委员会，研究在吉隆坡建立医学院的可能性。研究委员会在1961年的报告中建议在吉隆坡建立医学院，以每年为马来亚提供100名医生。 《大学章程》于1962年9月建立了医学院。

T.J.博士Danaraj于1日担负起第一位医学教授和学院院长的职责。 1963年2月，选定的40名医学预科生于1963年5月被录取并入学。这些先驱者在等待FOM基础设施发展的同时在科学学院接受了教授。 1964年5月，第一批64名学生（包括从医学预科课程升迁的学生）被录取为医学课程的一年级（临床前学年）。在学院成立后的五年内，大学医院（现为马六甲大学医学中心（简称UMMC）于5月5日由杨迪珀尔图·阿贡、杜里·扬·玛哈·穆利亚·塞里·帕杜卡·巴金达je下正式开幕。从那时起，UMMC已从最初的中央塔楼迅速扩展到各种现代和专用的塔楼，以满足医学教育，临床护理和生物医学研究的需要。通过新知识和新发现的出现，与国外和本地合作伙伴的合作研究使FOM和UMMC处于生物医学科学技术的最前沿。医学研究已成为FOM的第二项优势，并且在UMMC和医学系内部建立了多个卓越中心和研究中心。该学院还扩大了教学活动，在本科生和许多研究生专业中培训了高素质的医学和专职医疗人员。

2.愿景使命


成为医学教育的首要卓越中心；成为享誉世界的总理医疗中心，并提供高效、敏感和热情的质量政策，提供出色的保健，教育和研究计划；成为支持医学系研究需求的优秀组织，并实现世界一流的组织。

3.职业规划


卫生服务和公共卫生管理领域的应用研究/教学/实践岗位；学术界、研究、工业、政府和许多其他领域提供多样化的职业岗位；大学职位、政府和私营部门以及工业界专家。

4.课程结构


1）公共卫生博士课程共 84 个学分，包括以下两个部分：第 1 部分由总共 24 学分的课程组成，包括 一门三学分的必修核心课程；一门六学分的必修实习课程；两门必修专业领域核心课程，每门三学分； 三门专业专业课程，每门三学分。 第 2 部分包括60 学分论文的研究：(a) 候选人必须先成功完成第 1 部分，然后才能进入第 2 部分，(b) 候选人应在必修核心课程中达到最低 B 级。

必修核心课程：研究协议开发 3 学分。必修实习课程：专业实习6学分。专业领域核心课程：公共卫生流行病学要点 3 学分、卫生政策和领导力 3 学分。专业专业课程：(1)卫生服务管理领域：医疗保健经济评价 3 学分、人力资源规划与管理 3 学分、卫生法与伦理学 3 学分、卫生经济学 3 学分、健康物流管理3学分、健康质量 3 学分；(2)家庭健康领域：妇女健康 3 学分、儿童和青少年健康 3 学分、终生健康 3 学分、营养与哺乳管理 3 学分、社会/行为与健康 3 学分；(3)环境健康领域：环境污染 3 学分、食品技术与健康 3 学分、废物管理 3 学分、人为因素和人体工程学 3 学分、残疾评估 3 学分、职业性肺部疾病 3 学分、OSH 管理系统 3 学分；(5)健康流行病学领域：高级流行病学 3 学分、临床流行病学 3 学分、传染病流行病学 3学分、非传染性疾病流行病学 3 学分；(6)生物医学统计学领域：比率和比例分析 3 学分、统计计算 3 学分、元分析导论 3 学分、临床试验原则 3 学分、健康研究中的定性方法 3 学分、流行病学批判性读物和专题 3 学分、营养流行病学 3 学分。

2）博士课程完全基于导师监督研究。 在候选人的学习结束时，必须提交论文，并且在论文成功通过答辩后，候选人将获得博士学位。 所有研究计划都必须得到学院的批准和监督。博士生论文字数不得超过10万字，必须在 12 个学期内完成学业。

3）医学博士（MD）学位候选人应提交一篇毕业论文（不超过100,000字），该论文必须是根据学院的推荐和审查员酌情审查的参议院批准的主题的原创作品，以审查员认为适合论文主题和相关科目的方式进行；除非获得参议院的批准，否则候选人不得早于其初始注册日期后二十四 (24) 个月或两 (2) 年内提交本论文。，候选人应至少三 (3) 个月以书面形式向注册主任发出其打算提交论文以供考试的通知。(a) 候选人必须在注册的前两 (2) 个学期内完成研究方法课程 (MVX8001)；(b) 在第二学期开始准备论文提案答辩。

5.专业介绍


1）公共卫生博士课程是为那些希望成为公共卫生实践领导者的人提供的高级专业学位，适用于那些打算在公共和私人卫生部门以及教学和研究组织中领导卫生组织的人，面向公共卫生领域的应用研究。每位选择此高级（博士学位）学位的候选人将继续攻读以下所提供的公共卫生专业之一：健康服务管理、家庭健康、环境健康、职业医学、健康流行病学、生物医学统计。学习计划包括两个部分（第 1 部分和第 2 部分）。第 1 部分包括包括实习课程在内的课程，第 2 部分包括导致论文的研究。除非事先获得学院院长的书面批准，否则候选人必须在开始第 2 部分之前成功完成第 1 部分。毕业生专注于公共卫生实践的个人的最高专业学位，一个高度结构化的计划，重点是所有五个核心公共卫生领域的课程工作，并保证候选人掌握知识和技能，并完成研究论文；培养学生成为公共卫生和卫生服务政策和实践的领导者。

2）博士（医学）研究方向包括：医学教育、医学物理学、护理学、公共卫生、医药科学、医学生理学、再生医学、健康研究伦理等。博士学位是马来亚大学首屈一指的研究培训学位。马来亚大学学术成就的顶峰，博士学位是根据在监督下准备的论文的考试授予的，这代表了对所选研究领域的知识和理解的原创和重大贡献。UM大学的博士课程由独立专家使用当前和国际公认的标准进行评判，旨在培养不仅具有现代技术知识和技能，而且非常多才多艺并能够在现实生活中处理问题的个人。博士学生将在他们的专业领域进行重点研究，这与当地的兴趣直接相关。此外，他们将通过媒体、研讨会、会议和主题演讲，以及通过研究附件和与国际公认研究中心的合作交流安排，敏锐地意识到国际舞台上面临的挑战。博士课程不仅仅是学术上的区别，这些计划旨在确保研究领域与当今的需求保持相关。同时，攻读博士学位的候选人预见到社会和行业标准、趋势和行为模式的变化；正是这些品质使他们成为杰出而有效的领导者。

3）医学博士（MD）计划为申请人提供在马来亚大学医学中心（UMMC）工作机会。马来亚大学医学院提供的医学博士课程是一个较高的博士学位课程，候选人必须在申请该课程之前已经具备必要的医疗经验。候选人运用先进的知识、理解和经验进行医学和健康研究，以战略性地管理和领导任何组织，通过终身学习，以道德和专业的行为传播研究成果和/或提供医学和健康研究方面的专家建议； 通过研究以创造性和创新的方式解决医疗和健康相关问题，以便能够有效地领导和沟通。毕业生通过研究人员和利益相关者之间的有效沟通和合作展示领导质量，使用批判性思维、解决问题和/或适当的科学技能来解决研究问题。
